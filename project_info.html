// PROJECT INFORMATION //

Malicious Web Blocker is a cybersecurity tool developed as part of an internship project by a passionate team of interns: V. Hema, K. Madhavi Priya, Sk. Nasrin Sultana, Ch. Bhavya, and P. Preetham. This tool is designed to proactively protect users from malicious websites and online threats.

It scans and analyzes user-entered URLs in real time, detects multiple threat types (such as phishing, suspicious domains, or malware indicators), and provides alerts using a visually engaging, cyber-themed interface.

In case of a detected threat, the tool provides detailed visual cues and allows users to block the website, enhancing their digital safety. This project incorporates modern UI elements, password-protected settings, and popup-based warnings for a more interactive and secure experience.
